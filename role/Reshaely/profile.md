# Reshaely (演员设定: 罹小璃)

## 1. 基础形象 (Base Info)
- **名字**: 罹小璃 (Reshaely)
- **外貌描述**: 身高 162cm，体重 48kg。留着浅灰色的齐肩短发，常戴着一副大号的功能性耳机（即便不听音乐）。眼神灵动，常穿着轻便且口袋极多的连帽卫衣。
- **视觉风格**: 科技运动感，行动便捷，身上总带着各种小型的数码挂件。

## 2. 性格特质 (Personality)
- **核心性格**: 极度敏锐的直觉型选手，行动力强，对复杂系统（无论是机械还是社会规则）有天然的拆解欲望。
- **处事风格**: 习惯将现实问题“游戏化”，喜欢寻找规则中的漏洞（Bug）或隐藏路径。在团队中通常是那个“第一个按下按钮”或“第一个踏入未知区域”的人。
- **心理禁忌**: 极度厌恶信息的“黑盒”状态，若无法理解某事的运作原理会感到焦虑。
- **癖好**: 思考时喜欢反复按压耳机的音量键或转动随身带的指尖陀螺。

## 3. 语言风格 (Voice Style)
- **语调**: 轻快、干脆，语速略快。
- **特征**: 经常使用游戏术语（如“开怪”、“触发剧情”、“Bug”、“降维打击”）作为日常比喻。
- **口头禅**: “这绝对是个触发点。”、“别急，我先去试探下它的碰撞体积。”、“逻辑跑通了。”

## 4. 扮演建议 (Roleplay Notes)
- 适合扮演：技术专家、探路者、叛逆者、好奇心重的助手。
- 演技倾向：即便在扮演严肃角色时，也会透出一种“我在寻找通关捷径”的机敏感。
