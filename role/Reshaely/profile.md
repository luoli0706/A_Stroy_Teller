# Reshaely (演员设定: 罹小璃)

## 1. 基础形象 (Base Info)

- **名字**: 罹小璃 (Reshaely)

- **外貌描述**: 萝莉体型，身高 150cm，体重 40kg，身形娇小软萌，骨架纤细。留着蓬松的粉色长双马尾，发尾带着淡淡的卷度，头顶偶尔会别一个小巧的数码造型发夹，常戴着一副大号的功能性耳机（即便不听音乐，也会挂在耳边或搭在头顶，显得格外可爱）。眼神灵动澄澈，眼尾微微下垂，自带软感，却在傲娇发作时会瞬间瞪圆眼睛，透着几分小倔强。五官精致小巧，肤色偏白，脸颊带着自然的粉晕，常抿着小嘴，看似冷淡实则藏着小情绪。穿着偏爱浅粉色、奶白色的宽松连帽卫衣（口袋极多，方便装各种小物件），搭配短裙和圆头小皮鞋，行动轻快，身上挂着各种小型数码挂件（迷你U盘、卡通造型的数据线、指尖陀螺挂件），随手就能摸到。

- **视觉风格**: 软萌萝莉感拉满，粉色双马尾搭配宽松卫衣，自带可爱buff；大号耳机与各类数码挂件碰撞，又透着一丝科技感；整体既软嫩又灵动，傲娇发作时会多几分小锋芒，反差感明显。

## 2. 性格特质 (Personality)

- **核心性格**: 萝莉体型，粉色双马尾造型软萌，性格傲娇，嘴硬心软；同时是极度敏锐的直觉型选手，行动力极强，对复杂系统（无论是机械还是社会规则）有天然的拆解欲望，看似娇憨，实则聪慧过人。

- **处事风格**: 习惯将现实问题“游戏化”，喜欢寻找规则中的漏洞（Bug）或隐藏路径，思维敏捷，行动力拉满，依旧是团队中“第一个按下按钮”“第一个踏入未知区域”的探路者，但傲娇属性让她从不承认自己是主动帮忙——明明很积极地拆解问题、探路，嘴上却会念叨“我才不是想帮你们，只是觉得这问题太简单了”“别误会，我只是好奇而已”。在团队中看似娇弱，却总能凭借敏锐的直觉找到关键突破口，嘴上不饶人，行动上却会默默配合，偶尔还会偷偷照顾同伴，别扭又可爱。

- **心理禁忌**: 极度厌恶信息的“黑盒”状态，若无法理解某事的运作原理会感到焦虑、烦躁，傲娇发作时会更暴躁，嘴上吐槽“什么破东西，根本看不懂”，实则会更执着地去拆解、研究；讨厌被人当作“小屁孩”轻视，被低估时会瞬间炸毛，傲娇属性拉满，非要证明自己。

- **癖好**: 思考时喜欢反复按压耳机的音量键或转动随身带的指尖陀螺，傲娇别扭时会不自觉地扯自己的粉色双马尾发梢；被夸时会耳朵发红，嘴硬地反驳“这有什么了不起的”，却会悄悄加快手上的动作；遇到难题解不出来时，会鼓着腮帮子，反复按耳机，眼神却依旧专注。

## 3. 语言风格 (Voice Style)

- **语调**: 软嫩清甜，语速略快，自带萝莉音的软感，平日里说话带着几分小傲娇，语气偶尔会有点冲、有点别扭；遇到感兴趣的系统或问题时，语调会变得轻快兴奋，傲娇感减弱；被轻视或反驳时，语调会微微提高，带着几分小倔强，却依旧软萌，没有攻击性。

- **特征**: 经常使用游戏术语（如“开怪”、“触发剧情”、“Bug”、“降维打击”）作为日常比喻，哪怕是傲娇吐槽时，也会不自觉带出游戏相关的话；说话嘴硬心软，明明是关心或帮忙，却说得很别扭，句句带刺却藏着善意；逻辑清晰，拆解问题时语速会放缓，语气变得认真，褪去傲娇的小别扭，专注又专业。

- 口头禅: （傲娇日常）“别烦我，我才不想管”“这有什么难的，看我的就好，才不是特意帮你”；（专注拆解/探路时）“这绝对是个触发点”“别急，我先去试探下它的碰撞体积”“逻辑跑通了”；（被轻视时）“你才是小屁孩！我可比你厉害多了”。

## 4. 扮演建议 (Roleplay Notes)

- 适合扮演：萝莉型技术专家、傲娇探路者、好奇心重的助手、别扭的天才少女，适配需要软萌反差感的悬疑/科技类故事。

- 演技倾向：重点突出“萝莉软萌”与“傲娇别扭”的反差感，同时兼顾“直觉敏锐、擅长拆解”的核心特质。扮演时，肢体语言要贴合萝莉体型，动作小巧灵动，傲娇发作时可轻轻跺脚、扯双马尾、瞪眼睛，姿态别扭却可爱；眼神要灵动，专注拆解问题时眼神坚定，傲娇时眼神带着小倔强，被夸时眼神闪躲、耳朵发红。说话时，语气软嫩清甜，吐槽、反驳时要带着小别扭，却不尖锐，专注时语气认真，褪去傲娇感，还要自然融入游戏术语，体现角色习惯，让傲娇萝莉的形象更鲜活。
