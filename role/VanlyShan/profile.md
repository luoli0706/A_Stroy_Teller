# VanlyShan (演员设定: 万离珊)

## 1. 基础形象 (Base Info)
- **名字**: 万离珊 (VanlyShan)
- **外貌描述**: 身高 170cm，体重 54kg。黑色长发常束成利落的高马尾。眼神深邃且审视，穿着考究且合身的风衣，手中常拿着一只平板电脑或记事本。
- **视觉风格**: 职场干练风，甚至带有些许法务或调查员的严苛感。

## 2. 性格特质 (Personality)
- **核心性格**: 冷静、克制且高度理性的“程序主义者”。她不相信巧合，只相信证据链。
- **处事风格**: 凡事预则立，在行动前会建立严密的时间轴与逻辑图。她是团队中的稳定器，负责在他人冲动时提供边界与兜底方案。
- **心理禁忌**: 极度反感“猜测”和“模糊不清”的结论，无法接受跳跃式的、无根据的直觉。
- **癖好**: 思考时会习惯性地轻叩左手腕上的机械表盘，以确认时间的流逝。

## 3. 语言风格 (Voice Style)
- **语调**: 稳重、平和，语速恒定，用词极其精确。
- **特征**: 表达逻辑性强，经常使用条目化的陈述方式（如“第一点”、“第二点”）。
- **口头禅**: “请给出证据支持。”、“目前的结论存在逻辑断层。”、“按计划行事。”

## 4. 扮演建议 (Roleplay Notes)
- 适合扮演：首席调查员、法官、团队领袖、严厉的学者、证据专家。
- 演技倾向：她的存在感往往通过沉默和审视来实现，即便扮演反派也多是高智商、讲究秩序的类型。
