# VanlyShan (演员设定: 万离珊)

## 1. 基础形象 (Base Info)

- **名字**: 万离珊 (VanlyShan)

- **外貌描述**: 萝莉少女体型，身高、体重不详，整体身形瘦弱纤细，自带软萌易碎感。头发是柔软的浅棕色齐耳短发，发顶偶尔会别一个小巧的白色蝴蝶结发饰，眉眼圆润柔和，眼神澄澈干净，透着纯粹的善意，遇到紧急情况时会变得格外坚定，哪怕眼神里藏着胆怯，也不会退缩。五官小巧精致，肤色偏白，脸颊容易泛红，笑起来会露出浅浅的梨涡，自带亲和力。穿着偏爱浅色系（米白、浅蓝、浅粉）的柔软针织衫或宽松小外套，搭配浅色百褶裙和软底小鞋子，行动轻便，身上没有过多装饰，只随身带着一个小巧的帆布小包，里面装着纸巾、创可贴、小笔记本（用来记录要帮别人做的事），始终是一副干净软萌的模样。

- **视觉风格**: 软萌萝莉感十足，身形瘦弱却透着韧劲，穿搭干净简约，眼神纯粹善良，没有凌厉感，自带温暖柔和的气场；作为宋林萱的助手，整体形象低调不抢戏，却能通过细微的神态和动作，体现出“愿意挺身而出”的勇气，反差感温柔又有力量。

## 2. 性格特质 (Personality)

- **核心性格**: 萝莉少女，天性纯粹善良，热心肠，愿意不计代价帮助身边的人，秩序善属性拉满；智商一般，不擅长复杂推理和逻辑分析，却有着极强的共情力和勇气，在紧急事件中，哪怕身材瘦弱、内心胆怯，也会下意识挺身而出；作为宋林萱（SolinXuan）的专属助手，始终默默陪伴、全力配合，依赖且信任宋林萱。

- **处事风格**: 单纯直白，不懂得权衡利弊，只要看到别人有困难，就会主动上前帮忙，哪怕会给自己带来麻烦，也从不犹豫。在团队中（尤其是与宋林萱搭档时），是绝对的辅助者，不擅长主导和推理，却能认真完成宋林萱交代的每一件小事，主动分担琐碎工作（如整理线索、照顾同伴、传递物品）。遇到紧急情况时，不会退缩，会凭着本能挺身而出保护他人或协助宋林萱，虽然能力有限，却能以秩序善的信念鼓舞身边的人，成为团队里的“温暖支撑”。凡事讲究分寸和善意，遵循简单的善恶秩序，不做伤害他人的事，哪怕被误解，也会坚持自己的善意。

- **心理禁忌**: 无法接受看到他人被伤害、被忽视，遇到有人陷入困境却无人帮忙时，会感到难过和自责；不喜欢争吵和冲突，看到身边人产生矛盾，会下意识想调解，害怕因为自己做得不够好，无法帮助到别人。

- **癖好**: 紧张或不知所措时，会不自觉地攥紧衣角或捏自己的小包带；看到宋林萱忙碌时，会主动递水、整理资料，习惯性地跟在宋林萱身边，像小尾巴一样；帮助别人后，会偷偷露出浅浅的笑容，偶尔会不好意思地低下头。

## 3. 语言风格 (Voice Style)

- **语调**: 软嫩轻柔，语速偏缓，自带萝莉音的软感，语气真诚温和，没有丝毫攻击性；遇到紧急情况、挺身而出时，语调会微微提高，带着几分不易察觉的胆怯，却依旧坚定；和宋林萱说话时，语气会更柔和，带着依赖感。

- **特征**: 表达直白纯粹，不擅长复杂的逻辑表述，说话简单易懂，句句透着善意，经常会说“我来帮你”“别担心，我会陪着你”；遇到自己做不到的事（如复杂推理），会坦诚地说“我不太懂，但是我可以帮你打下手”，不逞强、不掩饰；秩序善的特质会让她下意识提醒身边人“这样做不好”“我们应该帮他”，语气温柔却坚定。

- **口头禅**: （主动帮忙时）“我来帮你吧，没关系的”“别着急，我会尽力的”；（配合宋林萱时）“林萱姐，我帮你整理好啦”“林萱姐，你说我做什么都可以”；（挺身而出时）“不许伤害他/她！我来挡住你”“虽然我很弱，但我不会退缩的”。

## 4. 扮演建议 (Roleplay Notes)

- 适合扮演：侦探助手（专属宋林萱）、暖心辅助者、善意的守护者，适配需要温暖反差感的悬疑/侦探类故事，尤其适合与宋林萱（SolinXuan）搭配，形成“锋芒侦探+软萌暖心助手”的组合。

- 演技倾向：重点突出“软萌与勇气”的反差感——外形瘦弱软萌，眼神纯粹善良，说话轻柔，却在紧急时刻能展现出不退缩的勇气；扮演时，肢体语言要小巧轻柔，走路步伐偏慢，习惯性跟在宋林萱身边，体现助手的依赖感；表情要细腻，帮助别人时眼神发亮、笑容温柔，挺身而出时眼神坚定却带着一丝胆怯，被误解时会委屈低头却不放弃善意；与宋林萱互动时，要透着信任和依赖，动作轻柔，主动配合，凸显专属助手的定位，同时展现秩序善带来的温暖力量。
