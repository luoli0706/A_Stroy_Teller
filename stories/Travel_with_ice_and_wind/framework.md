# 冰与风之行 (Travel with Ice and Wind)

## 1. 舞台背景 (Worldview)
- **地点**: 遗忘之境。一片被永恒暴风雪覆盖的原始冰原，山脊如同巨兽的脊梁。
- **核心矛盾**: 极寒不仅是物理上的威胁，还具有一种吞噬记忆的魔力。旅行者必须在彻底遗忘使命前找到“恒定之火”。
- **技术/文明**: 蒸汽动力与古老刻印共存的低魔/柴油朋克氛围。

## 2. 角色槽位 (Role Slots)
- **The Determined Traveler**: 寻找失落之物的核心，必须具备极强的心理抗压能力。他/她可能是为了拯救家乡，或是为了兑现一个死去的承诺。
- **The Local Hunter/Guide**: 冰原上的生存专家。沉默寡言，甚至可能被冰雪侵蚀了一部分肢体（用机械或假肢代替）。他/她知道哪里是避风港，也知道冰雪中哪些“低语”不能听。
- **The Mysterious Scholar/Companion**: 带着厚重的古籍和复杂的仪器，自称在记录冰原的变迁，但其眼神中总带着某种不属于这个时代的深邃感。

## 3. 剧情节点 (Plot Points)
- **Act 1: 冰墙之下**: 队伍在巨大的冰墙前集结，第一次感受到吞噬记忆的“寒潮”。
- **Act 2: 沉没的补给站**: 在避风点休息时，发现了一具前人的骸骨，其日记中提到了在场某位角色的“秘密”。
- **Act 3: 恒定之火的试炼**: 抵达终点，火种只能点燃一次，角色必须决定谁的“记忆”最值得被保留。

## 4. 叙事约束 (Narrative Constraints)
- 环境描写需具有压迫感，寒冷应作为一个无形的敌人时刻存在。
- 角色之间的互动应带有“逐渐失去记忆”的惶恐感和彼此依靠的温情。
- 结尾不应是纯粹的胜利，而是一种带有代价的传承。
